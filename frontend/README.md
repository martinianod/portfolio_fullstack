# Portfolio Landing — Martiniano

Stack: **React + Vite + TailwindCSS 3.x**, i18n (ES/EN), proyectos desde `projects.json`.

## Requisitos
- Node.js 18+
- npm

## Cómo correr
```bash
npm install
npm run dev
```

Abre: http://localhost:5173

## Estructura
- `src/pages` — Secciones principales (Hero, Projects, Skills, Contact)
- `src/components` — Navbar, Footer, ProjectCard
- `src/data/projects.json` — Fuente de proyectos
- `src/i18n/i18n.js` — Traducciones ES/EN

## Build
```bash
npm run build
npm run preview
```
